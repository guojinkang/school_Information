# 2016级项目实训成果展示 -
 
## 《校园信息平台》- Java与大数据分析

### 项目简介

本项目主要是为了给同学们提供一个便捷、条理清晰的信息服务平台，为同学们收集学习和生活方面的信息，并通过一定的处理和汇总，以简洁明了的方式向同学们展示这些信息。
